# Security

AKMaestro is proprietary Murex-internal software.

Report suspected vulnerabilities privately to the repository maintainers using
the approved internal security-reporting channel. Do not open a public issue or
include credentials, customer data, source excerpts, raw Copilot prompts, tool
payloads, session identifiers, or audit logs in a report.

Include the affected AKMaestro version, operating system, Copilot surface and
version, minimal reproduction steps, and sanitized impact. The maintainers will
coordinate triage and disclosure through Murex security processes.

Only the latest development version is currently supported; the product has not
yet been released for team use.
